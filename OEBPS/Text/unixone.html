Abstract

’Unix One’ will take someone with no knowledge of Unix commands to being able to move confidently in a text terminal.

Through completion of class work the participants will be able to:

  • get information about the Unix system environment
  • refer the built-in help mechanisms
  • navigate directories in a file system
  • do simple file manipulations
  • edit and manupulate text
  • communicate with other shell users in the same system
